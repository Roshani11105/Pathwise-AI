# 🤖 Pathwise AI

# 📌 Overview
Pathwise AI is a smart guidance system designed to help users choose suitable career or academic paths based on their interests, skills, and performance. It provides personalized suggestions to help users achieve their desired goals.

# 🚀 Features
- User input-based recommendations  
- Suggests career or academic paths  
- Simple and interactive interface  
- Helps in decision-making  

# 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

# ▶️ How to Run

## 1. Clone the repository
git clone <your-repo-link>  
cd <project-folder>  

## 2. Open the project
Open `index.html` in your browser  

# 📷 Output

## Sample Interaction

User Input:
- Interest: Technology  
- Skills: Basic Programming  
- Goal: Software Developer  

System Output:
- Suggested Path: Web Development / Software Development  
- Recommended Skills: HTML, CSS, JavaScript, Python  
- Guidance: Practice coding daily and build projects  

## Final Result
The system provides personalized guidance and suggests a clear path to help users achieve their target career or academic goals.
